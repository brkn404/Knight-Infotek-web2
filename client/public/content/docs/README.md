# Product Documentation

This directory contains comprehensive documentation for all Knight InfoTek products.

## Structure

- **[BlockchainDNA](./blockchain-dna/)** - Universal Identity & Provenance Chain
- **[Cha-Ching Analytics](./cha-ching-analytics/)** - Real-Time Compliance & Security Platform
- **[ZeroKnight30](./zero-knight-30/)** - Air-Gapped Security & Integrity Monitoring
- **[ChainGuardian](./chain-guardian/)** - Mempool Monitoring & Hash Verification

## Documentation Standards

All product documentation follows a consistent structure:

- `README.md` - Main index and overview
- `getting-started.md` or `quick-start.md` - Quick start guide
- `product-spec.md` - Product specification sheet
- `architecture.md` - System architecture (non-proprietary)
- `api/` - API documentation
- `faq.md` - Frequently asked questions
- `resources.md` - Additional resources (whitepapers, case studies, etc.)

## Maintenance

To add or update documentation:

1. Edit the appropriate markdown file in the product directory
2. Follow the existing naming conventions
3. Update the product's `README.md` index if adding new sections
4. Rebuild the site: `npm run build`
